# B2B-Platform

A scalable automation ecosystem built with **Laravel 10** & **PHP 8.2**. This platform integrates:

- **Multi-vendor E-commerce:** Robust marketplace capabilities.
- **Automated CRM:** Streamlined customer relationship management.
- **Social Media Scheduling:** Plan and automate posts across platforms.
- **Workflow Engine:** A rule-based "IF-THEN" engine for custom business logic.

## Technical Stack
- **Framework:** Laravel 10
- **Language:** PHP 8.2
- **Architecture:** Modular & Scalable

## Getting Started
1. Clone the repository.
2. Run `composer install`.
3. Copy `.env.example` to `.env` and configure your database.
4. Run `php artisan migrate`.
5. Run `php artisan serve`.
